# Portfolio – elwindria.com

Ce portfolio est un projet que j’ai réalisé dans le cadre de ma formation chez **OnlineFormaPro**.
Il a pour objectif de centraliser et présenter la majorité de mes travaux.

Accessible en ligne :
👉 https://www.elwindria.com

---

## Technologies utilisées

Ce projet a été développé avec des technologies simples et fondamentales du web :

* HTML
* CSS
* PHP

---

## Objectifs du projet

* Mettre en pratique les bases du développement web
* Concevoir une interface claire et accessible
* Présenter mes projets et compétences
* Disposer d’un support professionnel consultable en ligne

## Contenu

Le site regroupe notamment :

* Mes différents projets réalisés pendant la formation
* Des exemples concrets de développement front et back
